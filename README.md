# Global Income Distribution & Inequality Analysis Platform

## Project Overview
This project is an interactive data analytics platform that analyzes global economic indicators such as GDP per capita, income inequality (Gini Index), and unemployment rates across multiple countries and years.

It helps in understanding global inequality trends and provides insights through interactive visualizations.

---

## Live Application
-> https://global-income-inequality-visualization-dashboard.streamlit.app/

---

## Tools & Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- Excel

---

## Features
- Interactive dashboard with filters (Country, Year, GDP Category, Inequality Level)
- KPI cards for key metrics
- Country-wise comparison
- Trend analysis over time
- Data exploration module
- Downloadable filtered dataset
